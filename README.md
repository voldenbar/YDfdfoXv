# 项目简介

## 核心功能
- 基于 YOLOv8 实现塑料分类检测，支持识别 6 类塑料：HDPE、LDPE、PETE、PP、PS、PVC
- 提供完整源码、训练好的模型、4628 张标注数据集和项目运行教程
- 支持多种输入方式：单张图片、图片文件夹、视频文件和本地摄像头
- 界面功能包括：实时显示目标位置、目标总数、置信度信息；支持自定义置信度和IOU阈值；可选择显示标签和原图；支持批量检测并查看所有结果；可选择检测模型；支持检测结果保存

## 适用场景
- 塑料回收分类自动化处理
- 垃圾分类检测站
- 塑料制品质量检测
- 环保监测系统
- 工业生产线塑料分拣

## 技术特点
- 采用 Pycharm+Anaconda 或 VSCode+Anaconda 开发环境
- 基于 Python 3.9，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等技术栈
- 提供完整的 UI 界面源文件，包含图标资源
- 模型已训练完成，配置环境后可直接使用
- 支持实时检测和批量处理两种工作模式
